# Credit Card Fraud Detection using Machine Learning

## Project Overview
This project detects fraudulent financial transactions using Machine Learning techniques.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Power BI
- SQL
- Streamlit

## Machine Learning Model
- Random Forest Classifier
- SMOTE for class imbalance handling

## Performance
- Accuracy: 98%
- Fraud Recall Improved to 77%

## Dashboard Features
- Total Transactions
- Fraud Cases
- Fraud Rate %
- Fraud by Merchant Category
- Fraud by Transaction Hour
- Foreign vs Domestic Fraud

## Business Insights
- Foreign transactions have higher fraud risk
- Certain merchant categories show more fraud attempts
- Specific hours have higher fraud activity
